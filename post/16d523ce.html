华科王者赛

发表于 2023-10-29 更新于 2023-10-30 本文字数： 1.5k 阅读时长 ≈ 3 分钟

10月27号-10月31号

听闻沛公要在这周六打王者菁英杯比赛，恰巧学校里面开校运会放假，于是直接就定了高铁前来观战。

10月27号

早上七点半起床去赶火车，由于学校附近的地铁站离得比较远，去的路上耽误了好一会，最后差几分钟就没赶上火车。

中午11点多在汉口下站，接着做了一小时地铁去在华科附近订的酒店。

还好酒店是提前一天预定的，不然就没房间了。

晚上一个人睡，十分孤独


被三人趟过的床

到酒店后睡到差不多五点，在南三门门口沛公来接我了，接着去找蒜，找教室半天没找到，但碰巧在教室一楼碰到了

之后就直接去吃烤肉了

)

我已经这么肥了？（蚌埠租赁

)

这是我们觉得吃过的最好吃的一家烤肉

吃完后会学校，商量第二天安排：下午去看比赛

之后就回房休息了

10月28号

昨天晚上睡得很晚，早上起不来，就直接到11点过起床，被蒜接到食堂吃饭

吃完饭直接就去看比赛了，现场是露天的，天气也还不错，接下来就看云烟表演了

刚到的时候还有一小时开始比赛

拍拍拍

沛公叫我们入场就座

赛前列队宣言

比分从2：0被扳到2：2时，我们心里都捏一把汗：

赢了，终于松了一口气：

晚上去吃了汉堡王，接着就打道回府了

10月29号

昨晚又是睡得很晚，今早十一点起来，发现联系不上沛公，就直接先叫蒜接我去食堂吃大饭了

吃饭的时候遇见一个在食堂做作业的女生，我还是第一次见在食堂还做作业的学生，长见识了。

差不哦十二点还是没沛公消息，猜测沛公应该是通宵了，本打算去沛公宿舍找找，但记不得宿舍号了（这下不得记一记？

于是就直接让蒜来我酒店等消息。

于是就打了差不多三小时王者，在五点多商量好去肥肥虾庄吃饭。

好久没吃过小龙虾了，这次直接吃了个够

这个土豆味道确实很不错：

清蒸肥虾？记不清楚名字了，前两道虾我忘记拍照了：

)

接着又吃了一碗脆骨炒饭，结完帐后沛公又去点了生蚝吃，蒜没点到蜜雪冰城，对此他表示这个晚上差一点就完美了：

我在吃肥肥虾的时候听到沛公说赢了比赛本来想找我们合照的，结果我们已经走了，这时我心里是觉得有些遗憾的，因为当时太饿就打算直接走了，又想到沛公不和我们一起吃饭，就招呼也没打直接走了。于是我就想三个人在一起再拍张照，吃完饭就都回到了我的酒店：

中途沛公说月亮很圆，恰好这时轻轨路过，我们都停了下来，我也顺手拍了张月亮照片，尽管画质很差，不过好歹能看出月亮真的很圆：

)

到酒店后我打算拿这个在比赛里象征性的牌子拍拍照，于是我们就都坐在床上，用沛公的手机拍了照：

照片还在沛公那里，这里先凑个数：

接着就都打算回去了，但我对于离别是很不擅长的，也是不喜欢的，看着他们要走了，我的身体自顾自地也背起来了书包，想要一起出去，嘴里也说着要去华科逛逛，其实只是再多想在他们身边待会。回学校的路上我和沛公走着，蒜也慢慢的骑着车等着我们，中途有段时间看不见蒜了，以为他跑了，结果发现只是我没看见而已（呵呵

进入校园后，沛公和我找了小电动，骑上车后，蒜问我这么安排，这时候我脑袋晕乎乎的，就说我去到处乱逛逛。这时沛公说去找个地方拍合照，这下正合我意，开冲！

到了校史馆门口，发现好黑，拍照估计看不清楚，

于是，亮堂堂的图书馆是个不错的想法。

在图书馆逛了一会，最终由蒜儿决定，在某个有校徽校训的前台前拍照，某路人姐姐给我们排拍了张合照：

没有在比赛后合照的遗憾终于被弥补了。

之后，我们互相道别，蒜送我回去，沛公回宿舍，路上很冷，沛公和我应该都在打摆子。

最后，我回到了酒店门口，蒜骑着宝马走了，我坐着电梯，走过这氛围有些诡异的走廊，回到了房间：

这次来武汉我每天都过得很开心高兴，不仅是见到了许久未见的兄弟们，也是突然间意识到，自己在大学生活的每一天，并不是孤独的，我们的心里总是住着一群人，默默地给予你力量与信心。
